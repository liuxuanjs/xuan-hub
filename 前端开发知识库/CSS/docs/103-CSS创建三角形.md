## 概述
使用CSS创建三角形是一个经典的技巧，主要利用border属性的特性来实现。这在制作箭头、提示框等UI元素时非常有用。

## 基本原理

### Border特性
当元素宽高为0时，border会形成四个三角形：
```css
.demo {
    width: 0;
    height: 0;
    border: 50px solid;
    border-color: red green blue yellow;
}
```

## 使用clip-path的现代方案

### 基本三角形
```css
.modern-triangle {
    width: 100px;
    height: 100px;
    background: #9b59b6;
    clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
}
```

### 各种形状
```css
/* 向右箭头 */
.arrow-right {
    width: 100px;
    height: 60px;
    background: #1abc9c;
    clip-path: polygon(0% 20%, 60% 20%, 60% 0%, 100% 50%, 60% 100%, 60% 80%, 0% 80%);
}

/* 钻石形状 */
.diamond {
    width: 100px;
    height: 100px;
    background: #e67e22;
    clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
}
```

## 浏览器兼容性

- Border三角形: 所有浏览器
- clip-path: Chrome 55+, Firefox 54+
- 伪元素: IE8+

## 性能考虑

### 减少重绘
```css
.triangle {
    will-change: transform;
    transform: translateZ(0); /* 开启硬件加速 */
}
```

### 避免复杂计算
```css
/* 推荐：使用固定值 */
.triangle {
    border-bottom: 10px solid #333;
}

/* 避免：复杂计算 */
.triangle {
    border-bottom: calc(1vw + 5px) solid #333;
}
```

## 总结

- Border方法是创建三角形的经典方案，兼容性最好
- clip-path是现代方案，更灵活但兼容性稍差
- 合理使用伪元素可以避免增加HTML标签
- 响应式设计时要注意三角形的缩放效果
- 在性能敏感的场景中，要考虑重绘和重排的影响
