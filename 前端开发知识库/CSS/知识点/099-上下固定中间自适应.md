## 概述
上下固定中间自适应是一种常见的页面布局模式，即页面头部和底部高度固定，中间内容区域自适应剩余空间。

## 实现方案

### 1. 绝对定位方案
```html
<div class="container">
    <div class="header">头部</div>
    <div class="content">内容区域</div>
    <div class="footer">底部</div>
</div>
```

```css
.container {
    position: relative;
    height: 100vh;
}

.header {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    height: 60px;
    background: #333;
    color: white;
}

.content {
    position: absolute;
    top: 60px;
    bottom: 50px;
    left: 0;
    right: 0;
    overflow-y: auto;
    background: #f5f5f5;
}

.footer {
    position: absolute;
    bottom: 0;
    left: 0;
    right: 0;
    height: 50px;
    background: #666;
    color: white;
}
```

### 2. Flexbox方案（推荐）
```css
.container {
    display: flex;
    flex-direction: column;
    height: 100vh;
}

.header {
    flex: 0 0 60px;
    background: #333;
    color: white;
}

.content {
    flex: 1;
    overflow-y: auto;
    background: #f5f5f5;
}

.footer {
    flex: 0 0 50px;
    background: #666;
    color: white;
}
```

### 3. Grid布局方案
```css
.container {
    display: grid;
    grid-template-rows: 60px 1fr 50px;
    height: 100vh;
}

.header {
    background: #333;
    color: white;
}

.content {
    overflow-y: auto;
    background: #f5f5f5;
}

.footer {
    background: #666;
    color: white;
}
```

### 4. calc()计算方案
```css
.header {
    height: 60px;
    background: #333;
    color: white;
}

.content {
    height: calc(100vh - 110px);
    overflow-y: auto;
    background: #f5f5f5;
}

.footer {
    height: 50px;
    background: #666;
    color: white;
}
```

### 5. table布局方案
```css
.container {
    display: table;
    width: 100%;
    height: 100vh;
}

.header,
.footer {
    display: table-row;
    height: 1px; /* 让内容决定高度 */
}

.content {
    display: table-row;
    height: 100%;
}

.header > div {
    height: 60px;
    background: #333;
    color: white;
}

.content > div {
    height: 100%;
    overflow-y: auto;
    background: #f5f5f5;
}

.footer > div {
    height: 50px;
    background: #666;
    color: white;
}
```

## 注意事项

### 1. 滚动问题
```css
/* 确保内容区域可以滚动 */
.content {
    overflow-y: auto;
    /* iOS滚动优化 */
    -webkit-overflow-scrolling: touch;
}
```

### 2. 兼容性处理
```css
/* 兼容不支持vh的浏览器 */
.container {
    height: 100vh;
    height: calc(var(--vh, 1vh) * 100);
}
```

### 3. 响应式处理
```css
@media (max-width: 768px) {
    .header {
        height: 50px;
    }
    
    .content {
        padding: 10px;
    }
    
    .footer {
        height: 60px;
    }
}
```

## 浏览器兼容性

- Flexbox: IE11+
- Grid: IE10+（需要前缀）
- calc(): IE9+
- vh单位: IE9+

## 总结

- Flexbox方案是最推荐的现代解决方案
- 绝对定位方案兼容性最好
- Grid布局适合复杂的网格结构
- 选择方案时要考虑浏览器兼容性和具体需求
