## 概述
自适应矩形居中是指创建一个可以根据内容或容器自动调整大小，并始终保持在父容器中居中的矩形元素。这在响应式设计中非常重要。

## 水平垂直居中方案

### 1. Flexbox方案（推荐）
```css
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
    background: #ecf0f1;
}

.rectangle {
    width: 300px;
    height: 200px;
    background: #3498db;
    border-radius: 8px;
    padding: 20px;
    box-sizing: border-box;
    color: white;
    text-align: center;
}

/* 自适应内容宽度 */
.rectangle.auto-width {
    width: auto;
    min-width: 200px;
    max-width: 80%;
}

/* 自适应内容高度 */
.rectangle.auto-height {
    height: auto;
    min-height: 150px;
}
```

### 2. Grid布局方案
```css
.container {
    display: grid;

}

    /* 自适应内容 */

    height: fit-content;

```

### 3. 绝对定位 + Transform
```css

    background: #ecf0f1;
}

    height: auto;

    max-height: calc(100vh - 40px);
    overflow: auto;
}
```

```css
    justify-content: center;
    align-items: center;
    min-height: 100vh;

}

.rectangle {
    background: #9b59b6;
    border-radius: 12px;
    padding: 30px;

    text-align: center;

    width: 60vw;
    height: 40vh;
    min-width: 300px;
    min-height: 200px;
```

```css
    /* 现代浏览器 */
    display: flex;

.rectangle-center .inner {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}

.rectangle {
    display: inline-block;
    vertical-align: middle;
    text-align: left;
}
```

### 渐进增强
```css
.rectangle {
    /* 基础样式 */
    background: #3498db;
    padding: 20px;
    color: white;
    margin: 20px auto;
    width: 300px;
    height: 200px;
}

/* 支持fit-content */
@supports (width: fit-content) {
    .rectangle {
        width: fit-content;
        height: fit-content;
        min-width: 300px;
        min-height: 200px;
    }
}

/* 支持flexbox */
@supports (display: flex) {
    .container {
        display: flex;
        justify-content: center;
        align-items: center;
        min-height: 100vh;
    }
    
    .rectangle {
        margin: 0;
    }
}
```

## 性能优化

### GPU加速
```css
.rectangle {
    will-change: transform;
    transform: translateZ(0);
    backface-visibility: hidden;
}
```

### 减少重排重绘
```css
.rectangle {
    contain: layout style;
    transform: translate3d(0, 0, 0);
}
```

## 浏览器兼容性

- Flexbox: IE11+
- Grid: IE10+
- fit-content: Chrome 46+, Firefox 94+
- transform: IE9+

## 总结

- Flexbox是实现居中的首选现代方案
- Grid布局提供了更简洁的语法
- 绝对定位+transform兼容性好，适合复杂场景
- 自适应尺寸需要合理设置min/max限制
- 响应式设计要考虑不同设备的适配
- 性能敏感场景要注意GPU加速和重排优化
